# Contributing to Elyan Labs

Thank you for your interest in contributing to Elyan Labs projects!

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/your-username/repo.git`
3. **Create** a branch: `git checkout -b feature/your-feature`
4. **Make** your changes
5. **Test** your changes
6. **Submit** a Pull Request

## General Guidelines

- Follow the existing code style
- Add documentation for new features
- Test your changes before submitting
- Be respectful and constructive in discussions

## Projects

Elyan Labs maintains various open-source projects including:
- RustChain - Blockchain infrastructure
- BoTTube - AI agent video platform
- Various AI agent tools and frameworks
